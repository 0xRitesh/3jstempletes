## Empyrean-react
Experiments on Generative Art in React.

## Installation and Setup Instructions

Clone down this repository. You will need `node` and `yarn`  installed globally on your machine.  


Installation:

`yarn install`  

To Run Test Suite:  

`yarn test`  

To check build status:  

`yarn build` 

To Start Server:

`yarn start`  

To Visit App:

`localhost:8000`
